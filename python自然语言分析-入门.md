### Python NLTK 自然语言处理入门与例程

> 自然语言处理（ NLP ）就是开发能够理解人类语言的应用程序和服务。

我们生活中经常会接触的自然语言处理的应用，包括语音识别，语音翻译，理解句意，理解特定词语的同义词，以及写出语法正确，句意通畅的句子和段落。

- NLTK 是一个当下流行的，用于自然语言处理的 Python 库。

- NLTK 安装

        pip install nltk
        
    使用nltk下载器来选择需要安装的软件包。
